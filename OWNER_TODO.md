## 📌 사용 방법

- 항목별 **소요 시간**과 **시작 기한**이 표시됨

- `[ ]` → `[x]` 로 체크하며 진행

## 🚨 즉시 시작 (이번 주 안에) — 외부 검수 시간 필요

### 1. Apple Developer Program 가입

- [ ] **가입 신청**
    - URL: https://developer.apple.com/programs/
    - 비용: $99/년 (약 13만 원)
    - 소요: 가입 즉시 결제, 승인 1~3일
    - 필요한 것:
        - Apple ID
        - 신용카드
        - 한국 사업자등록번호 (개인 사업자 가입 시) 또는 D-U-N-S 번호 (법인 가입 시)
    - **메모**:

- [ ] **D-U-N-S 번호 발급** (법인으로 가입 시만)
    - URL: https://developer.apple.com/support/D-U-N-S/
    - 무료, 5~14일 소요
    - 개인 사업자면 스킵
    - **메모**:

---

### 2. Google Play Console 가입

- [ ] **가입 신청**
    - URL: https://play.google.com/console/signup
    - 비용: $25 (일회성, 약 3.3만 원)
    - 소요: 24~48시간 승인
    - 필요한 것:
        - Google 계정
        - 신용카드
        - 본인 인증 (전화번호 / 신분증)
    - **메모**:

- [ ] **개발자 계정 인증 (DUNS 또는 신분증 인증)**
    - 2024년부터 신규 개발자 계정은 신분증 인증 필수
    - 사업자라면 사업자등록증
    - **메모**:

---

### 3. Kakao Developers 등록

- [ ] **개발자 계정 가입**
    - URL: https://developers.kakao.com
    - 비용: 무료
    - 소요: 즉시
    - 필요한 것: Kakao 계정
    - **메모**:

- [ ] **앱 생성**
    - "내 애플리케이션" → "애플리케이션 추가하기"
    - 앱 이름: 한미동맹단 (또는 결정한 이름)
    - 회사명: 운영 사업자명
    - 카테고리: "기타" 또는 "사회" (정치 카테고리 없음)
    - **메모**:

- [ ] **카카오 로그인 활성화**
    - 앱 설정 → 카카오 로그인 → 활성화 ON
    - Redirect URI 등록 (개발자가 설정해줄 것 - 일단 보류)
    - 동의 항목: 닉네임(필수), 프로필 사진(선택), 이메일(선택)
    - **메모**:

- [ ] **비즈니스 채널 신청** (앱스토어 출시 시 필수)
    - 사업자등록증 필요
    - 검수 기간 1~2주
    - **메모**:

---

### 4. Naver Developers 등록

- [ ] **네이버 개발자 센터 로그인**
    - URL: https://developers.naver.com
    - 비용: 무료
    - 기존에 등록한 앱이 있다면 그걸 사용
    - **메모**:

- [ ] **애플리케이션 등록 또는 점검**
    - "Application > 애플리케이션 등록"
    - 사용 API: "네이버 로그인" 체크
    - 환경: Android, iOS 둘 다
    - 패키지명/Bundle ID 등록 (개발자가 결정 후 알려줄 것)
    - **메모**:

- [ ] **검수 신청**
    - 비즈니스용 사용 시 검수 필수
    - 1주 이내 결과
    - **메모**:
